## IT/DevOps лексика для презентации

### Структура проекта / Architecture
- **die Architektur** — архитектура
- **die Komponente** — компонент
- **der Microservice** — микросервис
- **die Schnittstelle** — интерфейс (API)
- **die Datenbank** — база данных
- **das Lastenheft** — техническое задание

### DevOps процессы
- **bereitstellen** — разворачивать (deployment)
- **die Bereitstellung** — развёртывание
- **automatisieren** — автоматизировать
- **die Automatisierung** — автоматизация
- **überwachen** — мониторить
- **der Monitor** — мониторинг
- **konfigurieren** — конфигурировать
- **die Konfiguration** — конфигурация

### Infrastructure as Code
- **die Infrastruktur** — инфраструктура
- **der Code** — код
- **als Code** — as Code (IaC)
- **die Vorlage** — шаблон (template)
- **das Skript** — скрипт
- **ausführen** — выполнять

### Tools / Terraform / Ansible
- **die Ressource** — ресурс
- **der Anbieter** — провайдер
- **der Server** — сервер
- **virtuell** — виртуальный
- **der Container** — контейнер
- **die Umgebung** — окружение (dev/staging/prod)

### Проблемы / Risiken
- **das Risiko** — риск
- **das Problem** — проблема
- **die Herausforderung** — вызов/задача
- **die Sicherheit** — безопасность
- **absichern** — защищать

### Phrases для презентации
- *"Ich möchte Ihnen meine Lösung präsentieren..."*
- *"Die Hauptkomponente ist..."*
- *"Ich habe mich für diesen Ansatz entschieden, weil..."*
- *"Ein möglicher Nachteil wäre..."*
- *"Zur Automatisierung nutze ich..."*
- *"Die Sicherheit ist dabei gewährleistet durch..."*
- *"Haben Sie dazu Fragen?"*
